The Foundation - day 6

Posted by Kenzo Takahashi on Wed 06 April 2016

I did not do anything related to the Foundation. I learned React. It's been a year since I last wrote a web app. There are so much more resources for learning React than it was back then.

What my app does is it allows you to create an English sentence using GUI. The main goal of it is to teach grammar. The interface for the computer used to be just command line. So you typed everything. Now GUI is the primary way to interact with the computer. It effectively minimizes errors because only valid actions are available on GUI.

If we can do the same thing with English, that would be great. As long as you use the app, you won't make grammatical mistakes. But there is a catch: Unlike programming languages, English grammar is not well defined. So you can't express every possible English sentence using the app. I actually came up with this idea a while ago, but I gave up because of this reason. But now I think it doesn't have to be able to express every possible English sentence to be useful. It only has to be able to express a subset of English because this is just a learning tool. You are not going to use it for everyday writing. It would be much slower to write English than typing anyway.

I always wanted to build a grammar checker, but it is too hard to do it even with machine learning. Even if you could, it wouldn't be very useful for learning grammar. After proofreading hundreds of student essays, I realized that what we need to do is not fix what's already written, but prevent errors in the first place.

MS Word and Google Doc both have a basic spell & grammar checker. They are great for someone who already knows how to write English. It can catch subject-verb agreement, for example. But when I proofread something my students wrote, I often have to rewrite the whole sentence because it's too messed up.

It's rather hard to describe what my app does using just words. I hope I can build a prototype soon. It's going to be the most complex app I've ever built.
